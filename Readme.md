# TS проект на Koa для быстрого развёртывания

## Быстрый запуск:

```bash
$ yarn
$ yarn dev
```
После этого перейти на в браузере на `http://localhost:7777/test`, если всё ок, появится надпись "It works"

## Что входит:

- Koa
  - koa-bodyparser
  - koa-router
  - koa-static
  - @koa/cors
